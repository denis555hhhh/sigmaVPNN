🚀 БЫСТРЫЙ СТАРТ:

ШАГ 1: Скачайте БД с Railway
   - Откройте Railway Dashboard
   - Откройте Shell сервиса
   - Выполните: cat sigmavpn.db
   - Скопируйте содержимое и сохраните как sigmavpn.db

ШАГ 2: Запустите панель управления
   - Откройте SigmaVPN_Bot_Admin.exe
   - Нажмите "Файл" → "Открыть локальную БД"
   - Выберите скачанный файл sigmavpn.db

ШАГ 3: Просмотрите данные
   - Нажмите "🔄 Обновить"
   - Данные загрузятся в таблицы
   - Переходите между вкладками для просмотра

📊 КАК СКАЧАТЬ БД С RAILWAY:

СПОСОБ 1: Через Shell (рекомендуется)
   1. Откройте Railway Dashboard
   2. Откройте Shell сервиса
   3. Выполните: cat sigmavpn.db
   4. Скопируйте весь вывод
   5. Откройте Notepad и вставьте
   6. Сохраните как: sigmavpn.db

СПОСОБ 2: Через File Explorer
   1. Откройте Railway Dashboard
   2. Откройте File Explorer
   3. Найдите sigmavpn.db
   4. Нажмите Download

⚠️ ВАЖНО:

1. БД находится на Railway в корневой папке сервиса
2. Файл может быть большим - будьте терпеливы
3. Убедитесь, что скачали полный файл БД
4. Сохраняйте БД в безопасном месте
5. Не удаляйте БД на Railway!

═══════════════════════════════════════════════════════════════════════════════

❓ РЕШЕНИЕ ПРОБЛЕМ:

Проблема: "БД не найдена"
Решение: Убедитесь, что файл sigmavpn.db находится в папке с EXE

Проблема: "Ошибка при загрузке"
Решение: Проверьте, что это правильный файл БД (SQLite)

Проблема: "Данные не загружаются"
Решение: Нажмите кнопку "🔄 Обновить"

Проблема: "EXE не запускается"
Решение: Убедитесь, что Python установлен (для исходного кода)
