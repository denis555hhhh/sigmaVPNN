🧪 ТЕСТОВЫЕ СЦЕНАРИИ:

СЦЕНАРИЙ 1: Просмотр подписок
─────────────────────────────────
1. Нажмите /start
2. Нажмите кнопку "📦 Выбрать подписку"
3. Должны увидеть 3 подписки:
   - 🟢 Базовая (99 ₽)
   - 🔵 Стандарт (299 ₽)
   - 🟣 Премиум (799 ₽)

СЦЕНАРИЙ 2: Выбор подписки
─────────────────────────────────
1. Нажмите на одну из подписок (например, "🔵 Стандарт - 299 ₽")
2. Должны увидеть:
   - Описание подписки
   - Кнопку "💳 Оплатить через Yoomoney"
   - Кнопку "✅ Я оплатил"
   - Кнопку "⬅️ Назад"

СЦЕНАРИЙ 3: Оплата
─────────────────────────────────
1. Нажмите "💳 Оплатить через Yoomoney"
2. Откроется ссылка на Yoomoney с суммой платежа
3. Вернитесь в Telegram
4. Нажмите "✅ Я оплатил"
5. Должны увидеть сообщение об активации подписки

СЦЕНАРИЙ 4: Просмотр подписок
─────────────────────────────────
1. Нажмите "📊 Мои подписки"
2. Должны увидеть активную подписку с:
   - Названием плана
   - Статусом
   - Количеством устройств
   - Датой окончания

СЦЕНАРИЙ 5: Помощь
─────────────────────────────────
1. Нажмите "❓ Помощь"
2. Должны увидеть информацию о боте и контакты

🆘 РЕШЕНИЕ ПРОБЛЕМ:

Ошибка: "ModuleNotFoundError: No module named 'telegram'"
Решение: pip install python-telegram-bot==20.3

Ошибка: "Port already in use"
Решение: Закройте другие приложения или используйте другой порт

Ошибка: "Invalid token"
Решение: Проверьте TELEGRAM_TOKEN в telegram_bot.py

Ошибка: "Database is locked"
Решение: Закройте SQLite Browser и перезапустите бот
